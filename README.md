# Excel Data Split Tool（業務効率化）

膨大な行数のExcelデータを、条件ごとに自動で分割するツールです。

## 概要

フィルター・コピー・シート分割といった手作業を自動化し、
数時間かかる作業を数分に短縮します。

## 主な機能

* 条件別データ抽出
* シート単位での自動分割
* ワンクリック実行

## 対象ユーザー

* 調達・購買部門
* 事務・営業部門
* Excelでのデータ整理に時間がかかっている方
